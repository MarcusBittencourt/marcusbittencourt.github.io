Flying with pelican project

Install pelican

$ pip install pelican
$ pip install markdown
$ pip install typogrify

create pelican project based on quickstart template scafold

$ pelican quickstart

build html static pages and run on localhost:8000

$ cd /_project
$ make html
$ make serve
